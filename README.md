# 🏥 Sistema de Gerenciamento de Clínica Médica

Projeto acadêmico desenvolvido para a disciplina de **Programação Orientada a Objetos (POO)** da Universidade Cruzeiro do Sul.

O sistema simula o gerenciamento de uma clínica médica, permitindo o cadastro e gerenciamento de pacientes, médicos e exames, aplicando os principais conceitos de Programação Orientada a Objetos em Java.

---

## 📌 Objetivo

Aplicar na prática os conceitos estudados em Programação Orientada a Objetos, desenvolvendo uma aplicação organizada, reutilizável e de fácil manutenção.

---

## 🚀 Funcionalidades

- Cadastro de pacientes
- Cadastro de médicos
- Cadastro de exames
- Associação de especialidades médicas
- Menu interativo via terminal
- Manipulação de listas utilizando `ArrayList`
- Organização do sistema em múltiplas classes

---

## 💻 Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- IntelliJ IDEA
- Git
- GitHub

---

## 📚 Conceitos aplicados

Durante o desenvolvimento foram utilizados conceitos importantes de orientação a objetos, como:

- Herança
- Encapsulamento
- Sobrescrita e sobrecarga de métodos
- Classes e Objetos
- Construtores
- Abstração
- Polimorfismo
- Organização em pacotes
- Manipulação de coleções (`ArrayList`)

## ▶️ Como executar

1. Clone este repositório

```bash
git clone https://github.com/rkazuxj/clinica-java.git
```

2. Abra o projeto em sua IDE Java (IntelliJ IDEA, Eclipse ou VS Code).

3. Execute o arquivo:

```
Aplicacao.java
```
